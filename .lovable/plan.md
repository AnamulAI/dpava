
## DPA Virtual Assistant — Enterprise Home Page

### Tech Stack
React + Vite + TypeScript + Tailwind CSS + shadcn/ui components. Google Sans loaded via `@fontsource/google-sans` (or simulated via system fonts). All colors from the defined palette: Primary Navy `#0B1F3B`, Marine Teal `#1E6F7E`, Light Gray `#F4F6F8`.

### Design Principles Applied
- No gradients, no shadows, no rounded soft UI — flat enterprise feel throughout
- Spacious section padding (py-20/py-24)
- Typography hierarchy using font-weight and size, not color decorations
- All interactive elements have teal hover states
- WhatsApp CTA consistently uses `https://wa.me/88017414688828`
